# SQL Basics — תרגילים

## תרגיל 1 — CREATE TABLE

צור טבלת `products` עם:
- `id` — מזהה ייחודי, אוטומטי
- `name` — שם מוצר, חובה
- `price` — מחיר, עשרוני
- `stock` — כמות במלאי, ברירת מחדל 0

---

## תרגיל 2 — INSERT

הוסף 3 מוצרים לטבלה:
1. "Laptop", מחיר 2500, מלאי 10
2. "Mouse", מחיר 45, מלאי 50
3. "Monitor", מחיר 800, מלאי 5

---

## תרגיל 3 — SELECT עם תנאים

כתוב שאילתות:
1. שלוף כל המוצרים
2. שלוף מוצרים שמחירם מעל 100
3. שלוף שם ומחיר בלבד, ממוין לפי מחיר עולה
4. שלוף את המוצר הזול ביותר (LIMIT 1, ORDER BY price ASC)

---

1. עדכן את המחיר של Mouse ל-55

3. מחק מוצר שה-stock שלו הוא 0
